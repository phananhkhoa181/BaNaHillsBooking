## Image Optimization Tips

1. **Format**: Use WebP format for better compression (with JPG fallback)
2. **Compression**: Compress images to reduce file size while maintaining quality
3. **Resolution**: Use 2x resolution for retina displays
4. **Alt Text**: Add descriptive alt text for accessibility
5. **Lazy Loading**: Images are set to lazy load for better performance

## Temporary Placeholder Generation

For development purposes, you can use:
- Placeholder services like placeholder.com or unsplash.com
- Solid color backgrounds with text
- Free stock photos from Unsplash, Pexels, or Pixabay

## Image Replacement Instructions

1. Save your images with the exact filenames listed above
2. Place them in the `assets/images/` directory
3. Ensure images are optimized for web
4. Test on different screen sizes to ensure responsive behavior
5. Update alt text in HTML if needed

---
**Note**: All current placeholder references in the HTML will automatically work once you add images with these filenames.
